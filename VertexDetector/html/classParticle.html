Detailed Description

Particle class.

Used to initialise a Particle, which is always characterised by a Point and a Direction, which are its data members fPoint and fDirection. The third data member of a Particle is an integer label (fLabel), that can be used to keep track of the generated particles. The original position of a Particle is, usually, represented by a Vertex, in which a number of Particles equal to the Vertex's multiplicity are generated.
